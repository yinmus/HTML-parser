# Parser

**python код запускает localhost страничку**
___

## Зависимости: 
   
-  **Компилятор:**  **GCC**, **clang** (MinGW -Windows)
-  **Python 3.XX.x**

___
## Установка 
```bash
git clone https://github.com/yinmus/HTML-parser.git
```
**Компиляция**
```bash
make
```
**Запуск**

```bash
make run
```
```bash
python server.py
```
![scr2](.img/scr2.png)

**Очистка**
```bash
make clean
```
